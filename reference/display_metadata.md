# Display metadata for a `git2rdata` object

Display metadata for a `git2rdata` object

## Usage

``` r
display_metadata(x, minimal = FALSE)
```

## Arguments

- x:

  a `git2rdata` object

- minimal:

  logical, if `TRUE` only a message is displayed
