# Summary method for `git2rdata` objects.

Prints the summary of the data and the description of the columns when
available.

## Usage

``` r
# S3 method for class 'git2rdata'
summary(object, ...)
```

## Arguments

- object:

  a `git2rdata` object

- ...:

  additional arguments passed to `summary`
